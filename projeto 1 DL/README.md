# Census Income Analysis (Adult Dataset)

This project analyzes the Adult Census Income Dataset to understand the relationship between education, gender, and income level in the United States.

## Objectives

Main questions answered:

- What percentage of people without a college degree earn more than $50K?
- Among those people, what percentage are men and women?

## Dataset

Adult Census Income Dataset

Source:
https://archive.ics.uci.edu/ml/datasets/adult

Also available on Kaggle:
https://www.kaggle.com/datasets/uciml/adult-census-income

Dataset characteristics:

- 32,561 observations
- 15 features
- Mix of categorical and numerical variables

Target variable:

income
- <=50K
- >50K
